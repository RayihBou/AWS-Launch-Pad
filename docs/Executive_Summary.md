# AWS LaunchPad - Executive Summary

AI-powered toolkit that automates migration and modernization processes, starting as an internal SA productivity tool and evolving into a customer-facing solution that generates GenAI revenue on every migration engagement.

---

## The Problem

Solutions Architects spend significant time on repetitive migration tasks: analyzing inventories, calculating TCO, classifying workloads, generating proposals, and creating documentation. These tasks follow predictable patterns that AI can automate while keeping human oversight for critical decisions.

---

## The Solution: 6 Specialized AI Agents

### 1. Assessment Automator

Receives infrastructure inventories (Cloudamize, Migration Evaluator, or manual exports) and automatically generates a complete Business Case.

- Classifies workloads by migration strategy (rehost, replatform, refactor)
- Calculates TCO comparison: on-premises vs AWS
- Identifies dependencies and proposes wave planning
- Delivers a ready-to-present Business Case document

**Time saved:** from 2-3 days of manual analysis to under 1 hour.

---

### 2. Discovery Assistant

Processes the Business Case generated by Agent 1 and produces actionable technical recommendations tailored to the customer's environment.

- Analyzes the classified workloads and infrastructure profile from the assessment
- Maps each workload to the most suitable AWS services and architecture patterns
- Generates initial architecture proposals per migration wave
- Recommends industry-specific use cases based on the customer's tech stack and business context

**Time saved:** from 3-4 hours of manual architecture mapping to under 15 minutes.

---

### 3. Database Modernization Advisor

Analyzes the database landscape identified by Agent 2 and recommends optimal migration and modernization strategies for each database instance.

- Classifies databases by migration type (homogeneous vs heterogeneous)
- Recommends target engine (Aurora, RDS, DynamoDB, ElastiCache) based on workload patterns
- Evaluates licensing optimization (commercial to open source)
- Generates migration plans with effort estimates and downtime windows

**Time saved:** from 1-2 weeks of database assessment to under 4 hours.

---

### 4. DR/DRP Planner

Receives a server inventory and generates a complete Disaster Recovery proposal with multiple alternatives and cost estimates.

- Classifies servers by criticality (RPO/RTO)
- Proposes DR alternatives: Pilot Light, Warm Standby, Multi-Site, Hybrid
- Generates comparative cost tables with monthly estimates
- Delivers a ready-to-present DRP proposal document

**Time saved:** from 1-2 weeks of analysis to under 2 hours.

---

### 5. Code Modernization Advisor

Analyzes legacy application stacks and recommends modernization strategies with effort estimates.

- Evaluates current tech stack and identifies modernization paths
- Recommends target architecture (containers, serverless, microservices)
- Estimates effort, timeline, and required resources
- Generates technical design documents

**Time saved:** from 1-2 weeks of assessment to 1-2 days.

---

### 6. Post-Migration Documentation Generator

Automatically generates operational documentation after each migration phase.

- Creates operational runbooks
- Generates security and compliance documentation
- Produces post-migration validation checklists
- Updates architecture diagrams

**Time saved:** from 2-3 days of documentation to under 2 hours.

## Revenue Opportunity (Phase 4)

When customers use LaunchPad agents during their migration, they consume Amazon Bedrock tokens in their own AWS accounts. This generates incremental GenAI revenue on every migration engagement.

| Metric | Estimate |
|--------|----------|
| GenAI consumption per migration account | $500 - $2,000/year |
| With 20 migration accounts per year | $10,000 - $40,000 incremental ARR |
| Scales with territory growth | More migrations = more GenAI revenue |

Every customer doing a migration gets AI-powered tools that run on Bedrock in their account, turning migration engagements into GenAI adoption opportunities.

- Framework: Strands Agents SDK (open source, AWS)
- AI Engine: Amazon Bedrock (Claude, Nova)
- Deployment: Local (Phase 1-2), Amazon Bedrock AgentCore (Phase 3-4)
- Output: HTML documents with AWS Dark Theme
