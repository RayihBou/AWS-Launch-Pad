Analisis de Costos
  • Los tokens de Bedrock (Claude Sonnet 4.6) representan aproximadamente el 90% del costo total. Los output tokens ($15/1M) son 5x mas caros que los input tokens ($3/1M).
  • AgentCore Memory es el segundo componente de costo (~9%), principalmente por las operaciones de retrieval ($0.50/1K) y almacenamiento long-term ($0.75/1K records).
  • AgentCore Runtime cobra solo por CPU activo (I/O wait es gratis), lo que reduce significativamente el costo vs. compute tradicional.
  • Lambda, API Gateway, DynamoDB, CloudFront, S3, ECR y Cognito combinados representan menos del 1% del costo total.
  • El costo escala linealmente con el numero de mensajes. Duplicar mensajes duplica el costo de Bedrock y AgentCore.
  • AgentCore Identity no tiene costo adicional cuando se usa a traves de Runtime o Gateway.
Recomendaciones de Optimizacion
Modelo Alternativo
Usar Claude Haiku para consultas simples (~10x mas barato). Routing inteligente: Haiku para FAQ, Sonnet para analisis complejos.
Reducir Tokens de Input
Limitar historial enviado al agente (actualmente 20 msgs). Comprimir/resumir historial largo. Optimizar system prompt.
Batch Inference
Para reportes programados o analisis no interactivos, usar Bedrock Batch Inference (50% descuento en tokens).
Caching de Respuestas
Implementar cache para consultas frecuentes (pricing, documentacion). Reducir invocaciones a Bedrock para preguntas repetidas.
Memory Long-Term
Usar estrategia override ($0.25/1K) en vez de integrada ($0.75/1K) para reducir costos de almacenamiento a largo plazo.
Provisioned Throughput
Para uso alto sostenido, considerar Bedrock Provisioned Throughput (20-40% ahorro vs. on-demand).
Servicios de la Solucion
  • Amazon Bedrock (Claude Sonnet 4.6) - Modelo de lenguaje para el agente conversacional
  • Bedrock AgentCore Runtime - Ejecucion serverless del agente (Strands SDK, Docker arm64, facturacion por consumo activo)
  • Bedrock AgentCore Gateway - Conexion a 5 MCP servers (Knowledge, Pricing, WA Security, CloudWatch, CloudTrail)
  • Bedrock AgentCore Memory - Memoria a corto plazo (eventos de sesion) y largo plazo (hechos y preferencias)
  • Bedrock AgentCore Identity - Gestion de identidad (sin costo adicional via Runtime/Gateway)
  • AWS Lambda - Proxy (1 funcion) + MCP handlers (4 funciones)
  • Amazon API Gateway HTTP API - Punto de entrada con autorizador JWT de Cognito
  • Amazon DynamoDB - Historial de conversaciones multi-usuario (on-demand)
  • Amazon CloudFront - CDN para frontend React
  • Amazon S3 - Hosting de archivos estaticos del frontend
  • Amazon ECR - Registro de imagen Docker del agente
  • Amazon Cognito - Autenticacion con MFA TOTP obligatorio
